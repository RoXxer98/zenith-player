# Zenith — Local Music Player 🎧

Zenith is a local (offline-first) music player for Android focused on deep customization, performance and modern UI.

---

## 🌍 Available Languages

- Spanish
- English
- Portuguese (Brazil)

---

## 🎨 Customization Focus

Zenith gives full control over both music and UI:

- Change song covers, playlists and folders
- Edit song names and aliases
- Custom profile (name, image)
- Custom backgrounds
- Light, Dark and AMOLED themes
- Dynamic UI based on album artwork

---

## ⚙️ Tech Stack

- Kotlin + Jetpack Compose
- Material 3
- Media3 / ExoPlayer
- Room + Flow
- DataStore
- Coil
- Glance (widgets)

---

## 🚀 Features

- Crossfade between songs
- Audio visualizers (custom BeatDetector)
- Equalizer with presets
- Drag & drop queue
- Dynamic accent colors
- Favorites system
- Listening analytics
- Home screen widget

---

## 🎧 Offline First

Zenith works entirely with local files.

No streaming. No cloud dependency.

This ensures:
- Full privacy
- Offline usage
- Complete control over your library

---

## ⚡ Performance Focus

Built to maintain 60fps while handling:

- Real-time visualizers
- UI animations
- Crossfade engine
- Dynamic theming
